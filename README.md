- Free SEO audit tool
- Local SEO optimization

## Quick Start

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/arnoldwender/seo-duisburg-2026.git
cd seo-duisburg-2026

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# Development server runs at http://localhost:4321 (Astro) or http://localhost:5173 (Vite)
```

### Build

```bash
# Create production build
npm run build

# Output: dist/
```

### Preview

```bash
# Preview production build locally
npm run preview
```
