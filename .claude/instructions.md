# seo-duisburg-2026

## Project Overview

SEO Duisburg Website - SEO services for Duisburg & Ruhrgebiet region

**Live Site**: https://seo-duisburg.com

## Technology Stack

Astro, React, Tailwind CSS

## Project Structure

This project uses a standard structure with source files in `src/`.

Main directories:
- `src/`
- `public/`
- `.github/`

## Development

### Setup
```bash
# Install dependencies
npm install
```

### Development Server
```bash
# Start dev server
npm run dev
```

### Build
```bash
# Build for production
npm run build
```

## Key Features

- Built with Astro for optimal performance and SEO
- React components for interactive UI elements
- Tailwind CSS for utility-first styling
- Optimized for local SEO in Duisburg and Ruhrgebiet

## Development Guidelines

### Code Style
- Follow existing patterns in the codebase
- Maintain accessibility standards (ARIA labels, semantic HTML)
- Ensure responsive design across all breakpoints
- Test across major browsers (Chrome, Firefox, Safari)

### Commits
- Write clear, descriptive commit messages
- Reference issues when applicable
- Keep commits focused and atomic

### Testing
- Test all interactive features
- Verify accessibility with screen readers
- Check mobile responsiveness
- Run build before committing major changes
